# AI 国学智慧平台 - 企业家精神道场

## 📌 项目概述

**项目名称**：AI 国学智慧平台
**目标**：为高端企业家提供国学智慧与精神成长的高端服务平台
**核心定位**：精英阶层的精神道场，国学智慧赋能商业领袖

---

## 🎯 目标用户画像

**核心用户**：高净值企业家、企业高管、商业领袖
**用户特征**：
- 年龄：40-60岁
- 收入：年收入千万级以上
- 需求：精神成长、智慧修身、传统文化品位
- 痛点：工作压力大、寻求精神寄托、传统文化兴趣

---

## 🎯 核心功能模块

### 1. AI 国学智能体 🤖

**定位**：企业家的私人国学顾问

**核心能力**：
- 🧘 **面相分析** - AI 面相大师，结合传统相学与心理学分析
- 🏯 **道家智慧** - 道德经解读、养生之道、处世哲学
- 🔮 **风水顾问** - 办公风水、居家风水、商业布局建议
- 📜 **经典解读** - 论语、孟子、尚书等儒家经典 AI 讲解
- 🎯 **易经占卜** - 六爻、梅花易数等传统占卜
- 💼 **商业国学** - 曾国藩、王阳明等商圣智慧

**交互方式**：
- 自然语言对话
- 图片上传分析（面相、手相）
- 语音交互（高端体验）

### 2. 预约系统 📅

**预约类型**：
- 👨‍🏫 **一对一咨询** - 与国学大师视频/线下咨询
- 🏛️ **线下道场体验** - 参观高端国学文化会馆
- 🎓 **精品课程** - 小班制国学研修班
- 🧘 **禅修静心** - 禅修、太极、养生课程
- 🍵 **雅集活动** - 书法、古琴、茶道等文化雅集

**预约流程**：
1. 选择服务类型
2. 选择导师/时间段
3. 填写基本信息
4. 支付预约金
5. 确认预约成功

### 3. 会员体系 👑

**等级划分**：
- 🌱 **入门弟子** - 基础会员
- 📚 **书童** - 消费满 1 万
- 🎓 **儒商** - 消费满 10 万
- 🏆 **掌门** - 消费满 50 万
- 👑 **宗师** - 消费满 100 万（顶级会员）

**会员权益**：
- 专属客服
- 优先预约
- 私人国学顾问
- 高端线下活动邀请
- 定制礼品

## 🛠️ 技术架构

### 前端技术
- **框架**：Next.js 14 (App Router)
- **语言**：TypeScript
- **样式**：TailwindCSS + Shadcn/UI
- **UI 风格**：高端大气、文化底蕴、极简奢华
- **配色**：中国红 + 墨黑 + 金色

### 后端技术
- **运行时**：Next.js API Routes
- **ORM**：Prisma
- **认证**：NextAuth.js (手机号 + 微信)

### 大模型（豆包）🌶️
- **模型**：Doubao-pro-32k / Doubao-pro-128k
- **提供商**：火山引擎
- **集成方式**：REST API
- **应用场景**：
  - 国学对话
  - 面相分析
  - 风水建议
  - 经典讲解

### 数据库
- **主数据库**：PostgreSQL
- **缓存**：Redis

### 部署
- **前端**：Vercel
- **后端**：Railway / Render
- **数据库**：Supabase
